Stránka funguje tak, že ako main prvok sú stopky, pod nimi sú 3 tlačidlá Lap Start a Reset. Po začaní stopiek môžme porovnávať "lapy" klikaním na Lap button. Reset button resetuje stopky. Taktiež je možné použiť klávesy na ovládanie tlačidiel (SPACE - štart/pauza, L - lap, R - reset)
